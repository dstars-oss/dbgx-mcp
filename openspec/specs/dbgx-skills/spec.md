# dbgx-skills 规范

## 目的
待定 - 由归档变更 add-dbgx-skills 创建。归档后请更新目的。

## 需求
### 需求:提供统一的 dbgx-mcp 操作技能
系统必须提供一个名为 `dbgx-skills` 的技能文档，统一描述 `dbgx-mcp` 的命令执行步骤与 TTD 导航流程。

#### 场景:技能入口可发现
- **当** 用户在仓库技能目录中查询可用技能
- **那么** 系统必须存在 `dbgx-skills` 技能目录及其 `SKILL.md`

### 需求:定义 TTD SeekTo 的默认规避流程
技能文档必须要求在 TTD 跳转场景中默认不直接执行 `dx ...SeekTo()`，并必须提供“查询 `TimeStart` 后使用 `!tt` 跳转”的替代流程。

#### 场景:执行 TTD 跳转
- **当** 用户需要跳转到某个 TTD 位置
- **那么** 技能必须指示先查询 `TimeStart`，再执行 `!tt <Sequence:Steps>`，并要求使用 `!tt` 验证当前位置

### 需求:明确本变更不要求测试
本次变更的任务规范必须声明该技能变更不要求执行自动化或手工测试。

#### 场景:生成实施任务
- **当** 任务清单被创建
- **那么** 任务内容必须明确标记“无需测试”
